# To do List TP - Lourdes Henriquez

Aplicación de gestión de tareas que permita crear, filtrar, editar y administrar una lista de pendientes.

- Crear, editar y eliminar tareas.
- Marcar tareas como completadas o pendientes.
- Filtrar tareas por estado (Todas/Completadas/Pendientes).
- Confirmación de eliminación mediante modal.
- Persistencia de datos usando `localStorage`.
- Diseño responsive para escritorio, tablet y móvil.

El proyecto está desarrollado en React

---

## Requisitos para clonar
- Node.js >= 18
- npm >= 9
